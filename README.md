# Warehouse Operations Dashboard
This dashboard tracks key performance metrics across multiple warehouse locations using dummy data for demonstration. It’s designed to showcase operational analytics, layout clarity, and business storytelling — all built in Google Sheet. 
##  What this dashboard shows 
- **Pick Line Volume** — total pick lines processed per warehouse 
- **Dimension Changes & Errors** — monthly trends and error rates
-  **Claims Analysis** — volume, rate %, and regional breakdown
-  **Slot Value Distribution** — stacked bar and pie charts by warehouse
-   **Internal Orders** — department-level breakdown
-  **Tote Destinations** — outbound flow visualization 
- **Monthly Trends**
— claims % of orders over time

##  Visuals included 
- KPI cards for:
- Total Pick Lines
- Total Dims Changes
- Total Dims Errors
- Total Research Claims
- Total Back Orders
- Average Claims Rate %

- - Charts:
- Bar charts (monthly trends, pick lines, slot values)
- Line chart (claims % of orders)
- Pie charts (slot values, claims by region, internal orders, tote destinations)
- Filters: - Month - Warehouse

- ##  Built with
- **Tool:** Google Sheet
- **Data:** Dummy warehouse operations data
- **Design:** Clean layout, color-coded logic, interactive filters

- ##  Note
-  This dashboard is a **sanitized recreation** of a real-world operational layout. All data has been anonymized and rebuilt using dummy values for portfolio purposes.
